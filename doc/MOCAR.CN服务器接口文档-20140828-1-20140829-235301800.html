接口定义

AccessToken接口

获取AccessToken

资源标识: /authority/token

方法: GET

请求响应示例：

GET /authority/token?credential=WX31429825 HTTP/1.1
Host: api.mocar.cn
Accept: application/json
HTTP/1.1 200 OK
Content-Type: application/json
Content-Length: 66

{
    "accessToken" : "126688deb863604b",
    "expiresIn" : 7200
}

路径参数：

无

查询参数：

名称 类型 可选 描述
credential string - 凭据

请求数据：

无

状态码：

状态码 含义
200 成功获取accessToken
403 凭据无效

返回数据：

类型 描述
object Token信息对象

注销AccessToken

资源标识: /authority/token/{token}

方法: DELETE

请求响应示例：

DELETE /authority/token/126688deb863604b HTTP/1.1
Host: api.mocar.cn
Accept: application/json
HTTP/1.1 200 OK

路径参数：

无

查询参数：

无

请求数据：

无

状态码：

状态码 含义
200 AccessToken已注销

返回数据：

无

数据格式

凭据

凭据是一个0-9A-Za-z的字符串，头部两个字节用来标识凭据类型，余下字符串为凭据内容。

微信CODE凭据

类型标识：

WX

凭据内容：

微信接口“https://open.weixin.qq.com/connect/oauth2/authorize”返回的CODE值。

示例：

WX31429825
